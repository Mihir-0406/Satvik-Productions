# Satvik Productions 🎬
### *Crafting Cinematic Experiences for the Next Generation*

**Satvik Productions** is a premium production house startup website designed to showcase high-end video production, digital marketing, and brand advertisement services. This platform offers a cinematic user experience with a focus on luxury aesthetics and seamless client interaction.

---

## 🚀 Features

- **Premium UI/UX**: A dark-luxury themed interface with smooth GSAP animations and responsive design.
*   **Service Showcases**: Dedicated sections for Video Production, Brand Advertisement, Digital Marketing, and Social Media Management.
*   **Dynamic Portfolio**: A curated display of iconic projects and storytelling campaigns.
*   **Automated Lead Management**: Integrated contact form with MongoDB storage and real-time email notifications via Nodemailer.
*   **Startup Ready**: Built with a scalable architecture to support growing business needs.

---

## 🛠️ Tech Stack

**Frontend:**
- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Icons**: Lucide Icons
- **Typography**: Google Fonts (Outfit & Inter)

**Backend:**
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Automation**: Nodemailer (SMTP Integration)

---

## 📁 Project Structure

The project is organized into two main modules for better scalability and maintenance:

```text
Satvik Production/
├── frontend/          # Client-side UI (HTML, CSS, JS, Assets)
├── backend/           # Server-side API (Node.js, Express, Models)
└── .gitignore         # Security rules for sensitive data
```
